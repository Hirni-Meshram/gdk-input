virtual gamepad_ptr gdk::input::context::get_gamepad ( const size_t  index)
pure virtual

returns a ptr to a gamepad

the order of gamepads is the order in which they were connected to the device the index refers to a poisition in that order. the gamepad ptr is guaranteed not to be null, even if there are less than N gamepads connected, such a gamepad will simply return 0 for all input polling.
