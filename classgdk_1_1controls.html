Detailed Description

provides a way to bind input from a gamepad, mouse, keyboard to a single binding name e.g: create a "Jump" binding that is non-zero when spacebar or gamepad button zero is pressed e.g: a "Look up" binding that is non-zero when up arrow or mouse axis y or gamepad axis 0 is interacted with. This also allows a game to easily support many types of gamepads.
