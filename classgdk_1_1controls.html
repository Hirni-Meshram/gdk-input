Detailed Description

provides a way to map raw input from a gamepad, mouse, and keyboard behind a single logical input e.g: create a "Jump" binding that is non-zero when spacebar or gamepad button zero is pressed e.g: a "Look up" binding that is non-zero when up arrow or mouse axis y or gamepad axis 0 is interacted with.
